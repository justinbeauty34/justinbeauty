🚀 Guide de démarrage rapide
1. Commencez par les photos
Allez dans Bannière principale et Galerie photos pour uploader vos vraies photos. Survolez l'icône ℹ pour voir le format idéal.
2. Vérifiez vos tarifs
Dans Tarifs, parcourez chaque onglet et corrigez les prix. Cliquez sur n'importe quel texte pour le modifier.
3. Personnalisez les textes
Chaque section du site est éditable. Les zones de texte contiennent déjà les contenus du site, prêts à être retouchés.
4. Sauvegardez toujours
Après chaque modification, cliquez sur Enregistrer les modifications en bas de page. Vos données sont conservées même après fermeture.
